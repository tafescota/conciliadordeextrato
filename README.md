# Conciliador de Extratos - Atualizado

Projeto separado em arquivos individuais para publicacao.

## Arquivos principais

- `index.html`: pagina principal na raiz.
- `assets/css/styles.css`: estilos do sistema.
- `assets/js/app.js`: regras, parsers, leitura dos extratos e exportacoes.
- `pages/index.html`: mesma aplicacao usando caminhos relativos para a pasta `pages`.
- `site_publico/index.html`: copia para publicacao quando a hospedagem pedir uma pasta publica.

## Como subir

Envie esta pasta inteira. O arquivo de entrada e `index.html`.

Nao precisa rodar build, npm install ou servidor. E um site estatico.
